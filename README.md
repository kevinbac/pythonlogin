
## 模拟登录一些常见的网站

主要基于以下的 Python 的第三 library 


1. [requests](http://www.python-requests.org) 处理登录
2. [pillow](https://github.com/python-pillow/Pillow) 处理验证码
3. [rsa](https://stuvel.eu/rsa) 处理加密问题
